Simple JS Animations

All of the animations on this page are available within this JavaScript File. Navigate to the corresponding function to find the code used to create each animation. All animations are created by Cody Maxie and are available for free use.

Div Animations:

Moves a 50 pixel block back and forth. movingSquare()

Resizes block width and height. resizeBlocks()

Bounces the block. bounceBlock()

Continually changes the border from side to side. alternatingBorders()

Canvas Animations:

Creates a simple gradient for the canvas. applyCanvasGradient()
